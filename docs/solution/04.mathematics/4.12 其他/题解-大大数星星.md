## 大大数星星

[XDOJ-1188-大大数星星](http://acm.xidian.edu.cn/problem.php?id=1188)

## 分析

`n`条直线分平面的最大区域数。

数列：`2, 4, 7, 11, ...`。

通项公式：$f_n=\frac{n(n+1)}{2}+1$。

## 实现

```cpp
#include <iostream>
using namespace std;

typedef long long LL;

int main () {
    LL n;
    while (cin >> n) {
        cout << n * (n + 1) / 2 + 1 << endl;
    }
    return 0;
}
```
