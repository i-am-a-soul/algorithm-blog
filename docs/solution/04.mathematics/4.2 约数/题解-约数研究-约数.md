# 约数研究

[洛谷-P1403-约数研究](https://www.luogu.com.cn/problem/P1403)

## 分析

$\lfloor \frac{n}{k} \rfloor$等于`1 ~ n`中`k`的倍数个数。

**枚举倍数，计算其约数个数**不如**枚举约数，计算其倍数个数**。

![](/img/0058.bmp)

## 实现

```cpp
#include <iostream>
using namespace std;

int main () {
    int n, res = 0;
    cin >> n;
    for (int i = 1; i <= n; ++ i) // 枚举约数
        res += n / i;
    cout << res << endl;
    return 0;
}
```
